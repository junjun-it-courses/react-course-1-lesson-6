бработать Pug в Gulp можно с помощью популярного плагина gulp-pug.
Заметили, что самые популярные плагины даже называются похожим образом? :)

Его установка и использование в точности повторяет подход, который
использовался при использовании плагина gulp-sass. Для использования
необходимо установить пакет pug и gulp-pug. Это можно сделать в одну строку:

npm install pug gulp-pug --save-dev

Так же обработка файлов происходит по уже знакомой нам структуре:
открытие файла, обработка, запись файла. Объединим с помощью функции
parallel функции в рамках одной задачи build:

Важно: в большинстве случаев в функциях-обработчиках, таких как pug()
и sass() из примеров выше, доступны дополнительные параметры, позволяющие
задать тонкие настройки компиляции. Всегда обращайте внимание на доступные
параметры
