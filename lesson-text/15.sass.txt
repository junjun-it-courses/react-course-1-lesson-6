Для обработки файлов, созданных при помощи препроцессора SASS
существует пакет gulp-sass. Это популярный пакет, который обновляется
вместе с обновлениями самого препроцессора. Для его использования
требуется установить два пакета:

sass — основной компилятор SASS
gulp-sass — плагин для Gulp
Установить всё можно одной командой:

> npm install sass gulp-sass --save-dev
